== Description ==

This is hero addone plugin for elementor pagebuilder. With this plugin, you can easily add awesome hero section into your website.

== Installation ==
1. Upload `hero-for-elementor.zip` to the `/wp-content/plugins/` directory and exctract
2. Activate the plugin through the 'Plugins' menu in WordPress
